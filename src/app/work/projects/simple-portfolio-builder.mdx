## Overview

Development of a responsiveness and functional web app with React + HeroUI + Render with AI chatbot integrated named "EventBot".

## Key Features

- **Component Library**: Built a set of modular, reusable UI components using React and styled-components with Tailwind, focusing on accessibility and responsiveness.
- **Theming and Customization**: Built a theme toggle for custom the light and dark theme in all the web app.
- **Events map**: Integrated an event map with different kind of event filters.
- **EventBot**: Built an AI chatbot with large training in events for childrens between 4 and 14 years old, with family-friendly language, and limited in 200 tokens.

## Technologies Used

- **React**: For fast, server-rendered React applications.
- **HeroUI**: For creating reusable components.
- **Tailwind**: For styling React components with a modular, themable approach.
- **MongoDB**: For building a maintainable data.

## Challenges and Learnings

One key was find the balance between the functionality and the responsiveness design.
